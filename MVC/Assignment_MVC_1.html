MVC Assignment 1: Game Rental Site

You will build a simple site to track games for a game rental store.

Part A: Project Setup

  1. Template: ASP.NET Core Web App (Model-View-Controller) C#
  2. Project name: MvcGame
  3. Solution name: MvcGame
  4. Configure for HTTPS: not selected

Part B: Game Class

Create a C# class to model our game data.

  1. GameId as an integer, which will be the Key
  2. Title as a string (required)
  3. Developer as a string (required)
  4. Genre as a string (required)
  5. ReleaseYear as an integer (required)
  6. PurchaseDate as a DateTime (not required)

Part C: Games Controller

  1. Scaffold an MVC Controller with views, using Entity Framework named GamesController
  2. Change the database to SQLite and name it games.db
  3. Do not seed the database programmatically
  4. Add at least 3 games to your database interactively

Part D: Layout

  1. Modify _Layout.cshtml and Home/Index.cshtml
    1. Set the title to your store's name (you can be creative)
    2. Place a logo, image, or slogan for your store on the home page
    3. Add an anchor Games to the page header that links to GamesController/Index

Part E: Submission

Submit a zip archive containing your project to the Canvas drop box.
