API Assignment

Today we will build an API Web Service and corresponding client MVC application to implement a simple shopping list.

API Project

  1. Name your project: ShoppingListApi
  2. Use this class for the Model 
        public class ShoppingListItem
    {
        int id;
        string? Item;
        bool Purchased;
    }
  3. Store the ShoppingListItem objects in a SQLite DB.
  4. Implement these methods
    • GET GetShoppingListItems() - returns a list of all ShoppingListItems
    • POST PostShoppingListItem(ShoppingListItem shoppingListItem) - adds an item
    • PUT MarkAsPurchased(int id) - set Purchased to true for one item
